# STAC PLANET API

Converter from STAC request to Planet request then from Planet response to STAC response.

## Development

### Prerequisites

- [uv](https://docs.astral.sh/uv/)

### Setup

```bash
make setup
```

### Running checks

```bash
make check
```

### Formatting

```bash
make format
```

### Building Docker image

```bash
make dockerbuild
```
